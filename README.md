## 🛠️ Como executar localmente

1. Clone este repositório:
   ```bash
   git clone https://github.com/GVerdans/portifolio-GVS.git
   ```
2. Acesse a pasta:
   ```bash
   cd portifolio-GVS
   ```
3. Abra o arquivo `index.html` no seu navegador:
   ```bash
   open index.html
   ```
   _Ou clique duas vezes no arquivo para abrir._
